## Core Implementation (UNCHANGED)

All core model implementations and inference algorithms remain exactly as in the original:

### Models
- **`polynomial()`**: Degree 2 polynomial prior
- **`point(x, curve)`**: Single observation model
- **`npoint_curve(xs)`**: Multi-point curve model
- **`npoint_curve_with_outliers(xs, ...)`**: Outlier detection model using Cond
- **`enumerative_gibbs_outliers(...)`**: Gibbs sampling for discrete outliers
- **`mixed_gibbs_hmc_kernel(...)`**: Combined Gibbs/HMC for outlier detection

### Key Features
- **Direct input pattern**: Models take `xs` as input to avoid static issues
- **Const wrapper**: Use `Const[int]` for static parameters
- **Outlier detection**: Natural mixture modeling with Cond combinator

## CLI Usage (SIMPLIFIED)

```bash
# Generate all 5 figures (default)
pixi run -e curvefit python -m examples.curvefit.main

# Generate only trace density figures
pixi run -e curvefit python -m examples.curvefit.main traces

# Generate only scaling analysis
pixi run -e curvefit python -m examples.curvefit.main scaling

# Generate only outlier detection comparison
pixi run -e curvefit python -m examples.curvefit.main outlier

# Custom parameters
pixi run -e curvefit python -m examples.curvefit.main scaling --n-trials 200
```

### CLI Arguments
- `mode`: Which figures to generate (all, traces, scaling, outlier)
- `--n-points`: Number of data points (default: 10)
- `--seed`: Random seed (default: 42)
- `--n-trials`: Trials for scaling analysis (default: 100)
